# Meowgram - a social network for cats

## Introduction
A simple social network pet-project made in React.

## Technologies
- Node.js 16.14.2
- TypeScript 4.9.5
- React 18.0.0
- Redux 8.0.4
- Redux-Saga 1.2.2

## Setup
To run the project locally, you can clone it and follow these steps:

1. Run `npm install`
2. Specify the `.env.local` file
3. Run `npm start`
